# Plan de Obtención de Requisitos - ReMedical

## Técnicas Planificadas

### 1. Entrevistas Estructuradas
- 12 médicos endocrinólogos
- 60-90 minutos por sesión
- Guía semi-estructurada

### 2. Observación Contextual
- 20 horas en 2 hospitales
- Observación directa consultas

### 3. Workshops Multidisciplinarios
- Médicos, enfermeras, administradores, desarrolladores
- 4 sesiones de 3 horas

### 4. Análisis Literatura Científica
- Papers sobre predicción diabetes con ML
- Guidelines ADA Standards of Care

### 5. Análisis Competitivo
- Evaluación 7 plataformas similares
- Identificación gaps y oportunidades

---

## Presupuesto

- **Tiempo estimado**: 340 horas-persona
- **Presupuesto**: $34,000 (10% del total)

---

## Formato de Entrega por Stakeholder

| Stakeholder | Formato |
|-------------|---------|
| Médicos | Prototipos, casos de uso |
| Pacientes | Lenguaje simple, FAQs |
| Desarrolladores | SRS, diagramas UML |
| QA | Casos de prueba, métricas |
| Reguladores | Evidencia documental, trazabilidad |
| Inversores | Roadmap, métricas de negocio |

## Riesgos a Gestionar

1. **Dataset insuficiente** → Alianza hospital universitario
2. **Reclasificación FDA** → Asesor regulatorio desde inicio
3. **Resistencia usuarios** → Explicabilidad + capacitación
4. **Variabilidad EHR** → Estándar HL7 FHIR
